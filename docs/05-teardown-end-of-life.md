# Teardown and End of Life

TreeGuard is designed to be temporary. A successful deployment ends with removal.

## When to remove

- Tree trunk is within 3 cm of frame edges and no larger ring is available
- Tree is 2+ years established and surviving without supplemental watering
- Tree has died (remove promptly — dead blinking boxes are litter)
- Electronics have failed and won't be repaired

## Teardown procedure

1. Power off electronics
2. Disconnect sensor probe — pull straight up gently, do not snap off in soil
3. Unclip electronics enclosure from spine
4. Unscrew frame segments (T10 security Torx)
5. Pull ground stakes
6. Bag all parts separately

## What to do with parts

| Part | Action |
|------|--------|
| Electronics module | Test, reflash, redeploy on next sapling |
| 18650 battery | Test capacity; if >70% original, reuse; otherwise recycle at battery drop-off |
| CR123A battery | Recycle at battery drop-off (do not trash) |
| PETG frame segments | If intact: reuse. If degraded: recycle (code 1) or return to maker group |
| Ground stakes | Reuse |
| Sensor probe | Test; reuse if functional, dispose as e-waste if not |
| Silicone, wires, screws | Reuse or recycle appropriately |

## Reporting

Log the teardown in your field log:
- Date removed
- Tree status: alive/dead/unknown
- Total months deployed
- Parts condition: reusable / degraded / failed
- Any notes for future deployments at this site

This data is critical for the project. It tells us actual survival rates.
