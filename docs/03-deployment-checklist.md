# Deployment Checklist

Use this checklist for every TreeGuard installation.

## Pre-deployment

- [ ] Read [00-before-you-plant.md](00-before-you-plant.md)
- [ ] Permission secured (written, if public land)
- [ ] Utility lines checked (811 / local equivalent)
- [ ] Species identified, correct preset loaded
- [ ] Sensor tested: dry reading in air, wet reading in cup of water
- [ ] LED test: red blinks, green blinks, amber blinks (run self-test mode)
- [ ] Battery charged (18650) or fresh (CR123A)
- [ ] Frame assembled and hardware secured

## Installation

1. Dig planting hole per species requirements
2. Plant sapling, backfill, water thoroughly
3. Position frame around sapling — **zero contact with trunk**
4. Drive 3 ground stakes at frame base, minimum 15 cm deep
5. Insert sensor probe 20-30 cm deep, adjacent to rootball (not through it)
6. Orient solar panel (V1) toward south (northern hemisphere) or north (southern hemisphere)
7. Attach QR code sticker to frame at eye level
8. Power on electronics module
9. Verify LED enters HAPPY state (green) after initial soil reading

## Post-installation

- [ ] Photograph the installation (for field logs)
- [ ] Log GPS coordinates, species, date, preset ID in [data/field-logs/](../data/field-logs/)
- [ ] Attach signage: "TreeGuard - plain water only - 1 bottle max" (see [signage templates](../mechanical/signage/))
- [ ] Set calendar reminder: monthly check-in for first year
- [ ] Register tree in app/web (if using connected version)

## Monthly check-in

- [ ] Tree alive and growing?
- [ ] Frame intact, no trunk contact?
- [ ] LED responding correctly? (water if dry, observe state change)
- [ ] Solar panel clean? (wipe with cloth)
- [ ] Sensor probe still seated at depth?
- [ ] Any vandalism or damage? Report via QR link.
- [ ] Frame size still appropriate? (expand if trunk approaching edges)

## End of life (year 3-4 typically)

- [ ] Tree established and self-sufficient? Remove frame.
- [ ] Disconnect and retrieve electronics module (reuse on next sapling)
- [ ] Remove ground stakes
- [ ] Frame parts: reuse, return to maker, or recycle (PETG = recycling code 1)
- [ ] Update field log: tree status, total lifespan of deployment
